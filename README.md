# 🌦️ Weather Pattern Visualizer

Historical weather data analysis and trend visualization using Python.

## 📌 Project Info
- **Batch:** June 2026 — AI InternsElite
- **Type:** Minor Project
- **Tech Stack:** Python, Pandas, Seaborn, Matplotlib, OpenWeatherMap API

## 📊 Features
- Monthly temperature trend analysis
- Rainfall distribution visualization
- Humidity pattern tracking
- Seasonal comparison charts
- Correlation heatmap between weather variables

## 🚀 How to Run
1. Open the notebook in Google Colab
2. Run all cells from top to bottom
3. (Optional) Add your OpenWeatherMap API key for live data
